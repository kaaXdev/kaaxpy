kaaXpy — Python Code Editor (beta1.0)
A lightweight, dark‑themed Python editor with a custom UI, splash screen, syntax highlighting, autocomplete, and an integrated run terminal — all built in pure Python with Tkinter.

✨ Features
kaaXpy includes a full set of modern editor features while staying compact and dependency‑free:

Startup
Custom fade‑in / fade‑out splash screen

Branding: kaaXpy beta1.0

Editor UI
Custom draggable titlebar (no system chrome)

Centered window on launch

Dark gray + neon green hacker theme

Multi‑tab support (open multiple files at once)

Line numbers

Status bar (line/column indicator)

Coding Features
Python syntax highlighting:

keywords

strings

comments

Auto‑pairing:

()

''

Auto‑indent on new lines

Autocomplete popup for Python keywords

Undo/redo support

Integrated Terminal

Open .py files
